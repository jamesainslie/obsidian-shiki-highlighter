### Added
- Initial plugin implementation
- Reading mode syntax highlighting using Shiki
- Live Preview mode support
- Theme switching (light/dark mode sync)
- Copy button functionality
- Line numbers support
- Lazy loading of language grammars
- Settings tab with theme and language configuration
- Support for 100+ programming languages
- 8 bundled themes (One Dark Pro, Dracula, Nord, etc.)
- Comprehensive test suite with Vitest
- GitHub Actions CI/CD pipeline
- Bundle size optimization
